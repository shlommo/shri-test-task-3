# Приложение для создания и редактирования информации о встречах сотрудников

## Установка
```
npm i
```


## Запуск клиента
```
npm run client
```
[http://localhost:3501](http://localhost:3501)


## Запуск сервера
```
npm i
npm run dev
```
[http://localhost:3000](http://localhost:3000)

Для сброса данных в базе:
```
npm run reset-db
